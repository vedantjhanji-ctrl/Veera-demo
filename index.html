How the replenishment engine works
Day 0 — Purchase recorded, consumption model initialized
Day ~20 — CRM predicts stock-out window (±3 days accuracy after 3 orders)
Day ~22 — WhatsApp nudge sent via Shiprocket Engage API
Day ~25 — If no action: email + push notification triggered
Day ~28 — Final SMS reminder with 5% "come back" discount
